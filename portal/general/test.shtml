Возможные причины:

  • Проблема: Возможно, сервер с этим IP адресом является открытым и анонимным прокси-сервером. Это означает, что любой пользователь интернета может им воспользоваться совершенно безответственно. Мы вынуждены запретить доступ через такие прокси-сервера, потому что они широко используются для атак на нас или на почтовые ящики наших пользователей. Администраторы таких прокси-серверов нарушают пункт 5. «Нормы пользования сетью» и пункт 4. «Основных принципов взаимодействия сети Яндекса с другими сетями».
    Совет: Попробуйте отключить прокси сервер в настройках вашего браузера. Как это сделать самостоятельно описано тут.

    Как отключить прокси-сервер самостоятельно

    • В Internet Explorer 4.* надо зайти в View (Вид) → Internet Options (Свойства обозревателя) → Connection (Подключение) и убрать флажок Access Internet using a proxy server («Использовать прокси-сервер»).

    • В Internet Explorer 5.* и 6.* надо зайти в Tools (Сервис) → Internet Options (Свойства обозревателя) → Connections (Подключение) → LAN Settings (Настройка локальной сети) и убрать флажок Use a proxy server («Использовать прокси-сервер»).

    • В Mozilla Firefox 1.5.* надо зайти в Tools (Инструменты) → Options (Настройки) → General (Основные) → Connection Settings (Параметры соединения) и выбрать там Direct connection to the Internet («Прямое подключение к Интернету»).

    • В Opera 8.* надо зайти в Tools (Инструменты) → Preferences (Настройки) → Advanced (Дополнительно) → Network (Сеть) → Proxy Servers (Прокси-серверы) и снять там все флажки.

    Возможно, прокси-сервер установлен принудительно у Вашего провайдера или системным администратором Вашей компании. Попробуйте воспользоваться этой ссылкой.

  • Проблема: Доступ может быть запрещен в случае нарушения Лицензии на использование поисковой системы Яндекса.
    Совет: Прочитайте внимательно Лицензию на использование поисковой системы Яндекса

  • Проблема: Ваш компьютер заражен вирусом или spyware (вредоносной программой), которая автоматически обращается к Яндексу.
    Совет:Рекомендуем вам проверить компьютеры на вирусы, работающие на данном IP адресе.
